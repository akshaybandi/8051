# 8051
8051 Micro controller Assembly Programs.
This repositary, gives you insight of 8051 Assembly coding.

Here I am using AT89C51 micro controller as a target device.

Features of AT89C51 micro controller.
  1. 128 bytes of RAM
  2. 4k bytes of ROM
  3. 4 I/O ports
  4. 2 timers
  5. 1 serial port
 
Files :
  1. Delay
  2. Duty Cycle
  3. Periperals
